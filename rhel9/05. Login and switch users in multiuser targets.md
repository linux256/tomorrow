## UNDERSTAND AND USE ESSENTIAL TOOLS
# 05. Login and switch users in multiuser targets

---

### I. SUDO Y SU

`sudo -i`

> Inicia login shell de usuario root, manteniendo variables de entorno y con
> UID=EID=0. No se emplea contraseña de root. El usuario que lo ejecuta debe
> pertenecer al grupo `wheel`

`su -`

> Inicia login shell de root, con variables de entorno limpias, solo con EID=0.
> Sí se emplea la contraseña de root, que debe existir.

`ssh root@localhost`

> Inicia login shell de root, ademas de cargar modulo PAM de systemd requerido
> para algunas configuraciones
