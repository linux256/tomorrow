## CONFIGURE LOCAL STORAGE
# 31. Add new partitions,logical volumes and swap non-destructively

---

### I. PARTICIONES DE SWAP

*Si Linux necesita liberar espacio RAM, puede pasar las páginas de memoria menos
usadas a almacenamiento en disco, empleando para ello espacio de SWAP. El
espacio de SWAP suele ser una partición dedicada, pero puede ser un fichero sin
perdida de rendimiento entre una u otra opción. Las particiones dedicadas a
información de SWAP se identifican con los tipos `82` en discos con particionado
MBR y con el tipo `8200` en discos con particionado GPT. Usan estos comandos,*

`mkswap /dev/sdb6`

> Da formato a la partición `/dev/sdb6` para emplearla completamente como SWAP

`swapon /deb/sdb6`

> Habilita la partición `/dev/sdb6` para que se emplee como espacio de SWAP

`echo "/deb/sdb6    none    swap    defaults   0 0" >> /etc/fstab`

> Añade a `/etc/fstab` la línea de configuración para montar la partición de
> SWAP automáticamente al arrancar el sistema. En este caso **NO** hay un
> directorio donde poder acceder al contenido de la partición

>

### II. FICHEROS DE SWAP

```
dd if=/dev/zero of=/swapfile bs=1MB count=1024
mkswap /swapfile
swapon /swapfile
echo "/swapfile  none  swap  defaults  0 0" >> /etc/fstab
```

>

### III. USO DE SWAP Y DESHABILITADO

`free -m`

> Muestra información del uso de la memoria RAM y por tanto del SWAP

`swapon -s`

> Muestra el uso que se hace de los dispositivos configurados para SWAP

`swapoff /deb/sdb6`

> Deshabilita la partición `/dev/sdb6` para emplearse como espacio de SWAP
