## CREATE SIMPLE SHELL SCRIPTS
# 15. Process output of shell commands within a script

---

### I. SHEBANG Y BASH

`#!/bin/bash`

> Fija `bash` como interprete de las órdenes del script.**El script corre en un
> subshell**

`#`

> Comentarios en el script, de el carácter `#` hasta final de la línea

`exit 0`

> Establece el código de salida 0, que es CORRECTO, TRUE, al shell padre. Si el
> comando `exit` no fija explícitamente el código de salida a retornar, se
> devuelve el código de salida del último comando ejecutado

`$?`

> Código de salida devuelto por el último comando/script ejecutado

`chmod +x script`

> Se activa permiso de ejecución a Usuario, Grupo y Otros, al fichero `script`

`bash script`

> Permite ejecutar `script` sin que este tenga permisos de ejecución

`bash -x script`

> Ejecuta `script` en modo DEBUG, sacando por pantalla cada línea que se corre

### II. SALIDA DEL SCRIPT

`comando &> fichero`

`comando >& fichero`

`comando >fichero 2>&1`

> Vuelca en `fichero` STDERR y STDOUT, donde se crea o se trunca `fichero`

`comando &>>fichero`

`comando >>fichero 2>&1`

> Añade a `fichero` la salida STDERR y STDOUT
